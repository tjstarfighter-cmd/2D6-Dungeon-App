# Help / Cheatsheet

Quick reference for keyboard shortcuts and shell-v2 gestures. More content (palm rejection, NEXT flow, room-gen, toast patterns) is in flight — see [Rules](/rules) for game rules.

## Keyboard shortcuts (desktop)

| Key | Action |
| --- | --- |
| `Esc` | Close any open modal, overlay, or wizard |
| `Cmd / Ctrl + K` | Focus the Tables search |
| `Cmd / Ctrl + 1 / 2 / 3 / 4` | Switch Sheet sub-tabs (Loadout / Magic / Pack / Lore) |
| `?` | Toggle this Help / Cheatsheet |
| `/` | Focus the active surface's primary search input |

## Gestures

- **Map zoom** — `Ctrl / ⌘ + wheel` or two-finger pinch.
- **Map pan** — drag with the Pan tool selected, or two-finger drag on touch.
- **Palm rejection** — when an active stylus is in range, capacitive touch is suppressed for the duration so your palm doesn't draw.

## Tip

The Help / Cheatsheet is also reachable from the `⚙` Settings menu in the desktop header.
